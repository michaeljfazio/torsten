## Source: ouroboros-network repo (IntersectMBO/ouroboros-network, main branch)

## N2N Versions (as of cardano-node 10.6.2)
- **V14** (numeric 14): Plomin hard fork, mandatory since 2025-01-29
- **V15** (numeric 15): Adds SRV DNS support
- Versions 7-13 are obsolete/commented out

## Handshake Version Data (v14+ CDDL)
```
nodeToNodeVersionData = [networkMagic, initiatorOnlyDiffusionMode, peerSharing, query]
networkMagic = uint .size 4  (word32)
initiatorOnlyDiffusionMode = bool  (true=InitiatorOnly, false=InitiatorAndResponder)
peerSharing = 0..1  (0=disabled, 1=enabled)
query = bool
```

## Handshake Messages
```
MsgProposeVersions = [0, {version_num => version_data, ...}]
MsgAcceptVersion   = [1, version_num, version_data]
MsgRefuse          = [2, refuse_reason]
MsgQueryReply      = [3, version_table]
```

## Version Negotiation (Acceptable instance)
- networkMagic must match exactly, else Refuse
- diffusionMode = min(local, remote) — more restrictive wins
- peerSharing = local <> remote (monoid combine)
- query = local || remote

## ChainSync Messages (CDDL)
```
[0]                    MsgRequestNext
[1]                    MsgAwaitReply
[2, header, tip]       MsgRollForward
[3, point, tip]        MsgRollBackward
[4, points]            MsgFindIntersect
[5, point, tip]        MsgIntersectFound
[6, tip]               MsgIntersectNotFound
[7]                    MsgDone
```

## BlockFetch Messages (CDDL)
```
[0, point, point]      MsgRequestRange (lower, upper inclusive)
[1]                    MsgClientDone
[2]                    MsgStartBatch
[3]                    MsgNoBlocks
[4, block]             MsgBlock
[5]                    MsgBatchDone
```

## TxSubmission2 Messages (CDDL)
```
[6]                            MsgInit
[0, blocking, ack_count, req_count]  MsgRequestTxIds
[1, txids_and_sizes]           MsgReplyTxIds
[2, txid_list]                 MsgRequestTxs
[3, tx_list]                   MsgReplyTxs
[4]                            MsgDone (only valid in blocking StTxIds)
```
- blocking = bool (true=blocking, false=non-blocking)
- ack_count, req_count = word16 (NumTxIdsToAck, NumTxIdsToReq)
- txIdAndSize = [txId, txSizeInBytes(word32)]
- In blocking mode: req must be >= 1; reply must be NonEmpty
- In non-blocking mode: ack or req must be non-zero; reply can be empty
- Lists use indefinite-length CBOR encoding

## KeepAlive Messages (CDDL)
```
[0, word16]    MsgKeepAlive(cookie)
[1, word16]    MsgKeepAliveResponse(cookie)
[2]            MsgDone
```
- Cookie = Word16, randomly generated by client
- KeepAliveCookieMismatch failure if response cookie != request cookie
- KeepAliveInterval = configurable DiffTime (no hardcoded default found in source)

## PeerSharing Messages (v14 CDDL)
```
[0, word8]             MsgShareRequest(amount)
[1, [*peerAddress]]    MsgSharePeers(peers)
[2]                    MsgDone
```
PeerAddress encoding:
```
IPv4: [0, word32, word16]          (tag, ip4_addr, port)
IPv6: [1, word32, word32, word32, word32, word16]  (tag, 4x ip6_words, port)
```
- amount = Word8 (max 255 peers per request)
- Protocol error to respond with more peers than requested

## Time Limits
- shortWait = 10 seconds
- longWait = 60 seconds
- waitForever = no timeout
- ChainSync StIdle: configurable (ChainSyncIdleTimeout default 3373ms)
- ChainSync StNext StCanAwait: shortWait (10s)
- ChainSync StNext StMustReply (untrusted): random 135-269s
- ChainSync StNext StMustReply (trusted): waitForever

## Ingress Queue Limits
- ChainSync: chainSyncPipeliningHighMark(300) * 1400 + 10%
- BlockFetch: max(10 * 2097154, blockFetchPipeliningMax(100) * 90112) + safety
- TxSubmission: maxUnacknowledgedTxIds * (44 + 65536) + 10%
- KeepAlive: 1280 + safety
- PeerSharing: 5760 (4 * 1440 TCP segments)

## Connection Limits
- acceptedConnectionsHardLimit = 512
- acceptedConnectionsSoftLimit = 384
- acceptedConnectionsDelay = 5 seconds
